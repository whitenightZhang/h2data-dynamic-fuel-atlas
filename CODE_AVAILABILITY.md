# Code Availability

This repository contains the model code and web atlas used to support the
associated renewable hydrogen-derived fuels analysis.

## Included

- `src/h2data/`: 8760-hour optimization functions for renewable hydrogen,
  green ammonia, e-methanol, and Fischer-Tropsch/e-SAF systems, plus static
  annual-average screening scripts.
- `scripts/prepare_web_data.py`: utility for building the compact web-ready
  JSON extracts used by the atlas.
- `docs/`: GitHub Pages-ready static website and processed web extracts.

## Not Included

Large raw meteorological, geospatial, technology-cost, and intermediate
optimization output files are not committed to the repository. They should be
supplied through the command-line paths documented in `README.md`.

## Solver Requirement

The 8760-hour optimization functions require Gurobi and a valid Gurobi license.
The static post-processing scripts do not solve Gurobi models but require the
processed input CSV files generated by the broader workflow.

## Reproducibility Boundary

The repository is intended to document and rerun the computational workflow
where the required input data and solver license are available. The web atlas
can be viewed directly from the committed `docs/` directory without rerunning
the full optimization workflow.
